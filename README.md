- [Sass](https://sass-lang.com/)
- [Bootstrap](https://react-bootstrap.github.io/)
- [LocalStorage]


---

## 📥 Como baixar o projeto
```bash

#Clonar repositório
$ git clone https://github.com/marcelafreire/tema-sistemas-teste

#Entrar no diretório
$ cd tema-sistemas-teste

#Instalar dependências
$ npm install

#Iniciar aplicação 
$ npm start

````
